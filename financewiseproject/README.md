# FinanceWise - Personal Finance Guide

FinanceWise is an interactive personal finance portal designed to help users understand money management, calculate budgets, track savings goals, and evaluate loans. 

## Features

- **Budget Planner**: Calculate monthly budgets, remaining balance/savings, and view visual alerts based on your spending.
- **Savings Goal Planner**: Estimate how many months are required to reach a specific financial goal with completion date forecasting and progress bar tracking.
- **Financial Calculators**:
  - **EMI Calculator**: Calculate monthly loan repayments, total payment, and total interest.
  - **Simple Interest**: Calculate interest earned and accumulated total.
  - **Compound Interest**: Calculate interest with configurable compounding frequencies (Annually, Semi-Annually, Quarterly, Monthly).
  - **50-30-20 Split**: Analyze monthly net income split into Needs (50%), Wants (30%), and Savings (20%).
- **Interactive Forms**: A client-side validated contact form with clean error feedbacks.
- **Global Actions**: Smooth scroll to top, scroll indicators, and active navigation menu highlights.

## Technologies Used

- **HTML5** & **CSS3** (responsive styling)
- **Bootstrap 5** (components & layout grid)
- **Bootstrap Icons** (typography iconography)
- **Vanilla JavaScript** (dynamic interactions, client-side calculations, form validations)

## Getting Started

Simply open `index.html` in any web browser to run the application locally. No build step or installation is required.
