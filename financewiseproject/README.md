# FinanceWise

FinanceWise is a simple personal finance landing project with pages for budgeting, savings planning, calculators, and contact information.

## Project Structure

- index.html
- budget-planner.html
- savings-planner.html
- calculators.html
- contact.html
- css/
- js/
- images/

## Features

- Clean landing page
- Budget planning page
- Savings planning page
- Finance calculators page
- Contact page

## Getting Started

Open the project in a browser by launching index.html.

## Development

You can edit the HTML, CSS, and JavaScript files in their respective folders to expand the project.
